Nodos fórmula

Los nodos fórmula son pixel shaders, es decir, las operacione se aplican a nivel de pixel. Se pueden realizar una gran cantidad de operaciones matemáticas. La documentación de GLSL se puede encontrar en: http://www.opengl.org/sdk/docs/manglsl/.

Variables propias

En Shadertool se pueden utilizar algunas variables que no pertenecen a GLSL para simplificar el uso de muchas operaciones.
  • color: Color final del pixel, de tipo vec4.
  • pos: Posición del pixel en la imagen, de tipo vec2.
  • size: Tamaño de la imagen, de tipo vec2.
  • time: Variable que se incrementa con el tiempo, de tipo float.
  • colN: Color del pixel de la imagen de entrada N, de tipo vec4. Ejemplo: col1, col2... Únicamente válido si hay imágenes de entrada
